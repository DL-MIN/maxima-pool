# MaximaPool

A simple serverside Maxima pool implementation to deal with requests from [moodle-qtype_stack](https://github.com/maths/moodle-qtype_stack).


## Requirements

- Python 3 *(tested with v3.7)*
- Gunicorn
- Flask
- Gnuplot
- Maxima


## Install

1.  Install or compile [Maxima](http://maxima.sourceforge.net)

2.  Install required packages

    *Debian 10 (buster):*  
    ```
    $ apt install python3-flask gunicorn3 gnuplot
    ```

3.  Clone git repository

    ```
    $ git clone git@github.com:DL-MIN/maxima-pool.git /opt/MaximaPool
    ```


## Configuration

**Important:** MaximaPool does not authenticate nor check the origin of a request. Therefore, bind the WGCI application and your Moodle to an internal network.

1.  Create user `maximapool`

    ```
    $ useradd --disabled-password maximapool
    ```

2.  Create *systemd* service `/etc/systemd/system/maximapool.service`.

    Replace IP address and amount of worker to match your requirements. The number of workers should not exceed the number of threads!

    ```
    [Unit]
    Description=MaximaPool
    After=network.target

    [Service]
    User=maximapool
    Group=maximapool
    WorkingDirectory=/opt/MaximaPool
    ExecStart=gunicorn3 --bind 192.168.0.1:8000 --workers 48 wsgi:app
    Restart=on-failure

    [Install]
    WantedBy=multi-user.target
    ```

3.  Create folders

    ```
    $ cd /opt/MaximaPool
    $ mkdir -p assets/maxima plots tmp
    ```

4.  Create `assets/maximalocal.mac`

    Copy the autogenerated Maxima config file from STACK Healthcheck in Moodle `https://moodle.tld/question/type/stack/healthcheck.php`.

    Replace given parameters as shown:  
    ```
    file_search_maxima:append( [sconcat("/opt/MaximaPool/assets/maxima/###.{mac,mc}")] , file_search_maxima)$
    file_search_lisp:append( [sconcat("/opt/MaximaPool/assets/maxima/###.{lisp}")] , file_search_lisp)$

    STACK_SETUP(ex):=block(
        MAXIMA_VERSION_NUM_EXPECTED:0,
        MAXIMA_PLATFORM:"server",
        maxima_tempdir:"tmp/",
        IMAGE_DIR:"%PLOT-DIR%",

    ...
    ```

5.  Copy STACK instructions to `assets/maxima/`

    Download all files from [moodle-qtype_stack/stack/maxima](https://github.com/maths/moodle-qtype_stack/stack/maxima) to above-mentioned path.

6.  Change ownership to `maximapool`

    ```
    $ chown -R maximapool:maximapool /opt/MaximaPool
    ```

7.  Enable and start service

    ```
    $ systemctl daemon-reload
    $ systemctl enable maximapool
    $ systemctl start maximapool
    ```

8.  Configure Moodle

    - `qtype_stack | platform` set to *Server*
    - `qtype_stack | maximacommand` set to *http://192.168.0.1:8000*
